# ro_interview_assignment
RO Interview for SDN Developers

Fetch and validate data from an API endpoint.

#### CI using Github actions
 * Github Actions Workflow triggered on each `push`
 * Runs all unittests:
   * `test_invalid_clusters.py`
   * `test_invalid_records.py`
   * `test_sort_records.py`
 * The test commands are run under `python 3.x`
 * To check it out, select the `Actions` tab and select the `CI` workflow
